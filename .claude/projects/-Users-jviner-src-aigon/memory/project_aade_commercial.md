AADE (Aigon AI Development Effectiveness) is the planned commercial product offering for Aigon.

**Why:** Aigon is currently free/OSS. AADE (token tracking, git signals, rework detection, amplification dashboard, AI coaching insights) is a natural value-add tier. No competitor serves individual developers in this space — all target teams/orgs. The AI insights layer has per-call API costs that justify a subscription.

**How to apply:** When building AADE features, consider the free/paid boundary. Data collection may stay free; analysis and AI coaching are the premium layer. A separate research topic (research-aade-commercial-gate) covers commercialisation strategy. Use "AADE" as the official name, "Amplification" as the user-facing dashboard label.

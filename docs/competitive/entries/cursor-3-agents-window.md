# Cursor 3 Agents Window

- **Slug:** `cursor-3-agents-window`
- **Tier:** A — direct competitor (multi-agent inside the IDE)
- **Vendor:** Cursor
- **License:** Closed source
- **Last verified:** 2026-04-27 (R44)

## What it is
Up to 8 parallel agents in worktrees or remote machines, tiled-pane UI, launched April 2026. Closest IDE-embedded equivalent to Aigon's parallel-competition story.

## Matrix cells (public 5)
- **Unit of work:** prompt
- **Source of truth:** chat history (per agent)
- **Multi-agent posture:** up to 8 parallel agents
- **Model flexibility:** curated (Cursor Composer + frontier models)
- **Pricing:** $20/mo Pro, overages, $200/mo Ultra

## Closest to Aigon on
Multiple agents racing on the same task. The IDE-side analogue.

## Where it beats Aigon
- Best-in-class inline completion (Composer is sub-second).
- Tightest IDE integration; tiled multi-agent panes inside the editor.
- Aggregated diff view across agents.
- Embedded browser testing.

## Where Aigon wins
- Vendor independence — Aigon runs Claude Code / Gemini / Codex / Cursor / OpenCode side by side; Cursor 3 is single-vendor.
- Markdown specs in git instead of per-agent chat history.
- Cross-vendor cross-agent evaluation as the merge gate.
- Free OSS + BYO subscriptions vs Cursor's platform fee.

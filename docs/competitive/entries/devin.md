# Devin 2.0

- **Slug:** `devin`
- **Tier:** D — autonomous cloud agent
- **Vendor:** Cognition
- **License:** Closed source
- **Last verified:** 2026-04-27 (R44)

## What it is
"AI software engineer" running in cloud sandbox. $20/mo + $2.25/ACU. 13.86% real-GitHub-issue resolution. 83% more tasks per ACU vs v1. Multiple "Devins" can run in parallel for large-scale migrations.

## Matrix cells (public 5)
- **Unit of work:** issue
- **Source of truth:** hosted workspace
- **Multi-agent posture:** multiple cloud "Devins" in parallel (same agent, replicated)
- **Model flexibility:** Cognition-locked
- **Pricing:** $20/mo + $2.25/ACU

## Where it beats Aigon
- Zero local setup — fully cloud-hosted.
- Fully autonomous on cloud sandbox.
- Browser tooling, integrations (Linear, Slack, Datadog).
- Best fit for large-scale automated migrations.

## Where Aigon wins
- Vendor independence — Devin is one vendor, one model. Aigon runs Claude / Gemini / Codex / Cursor / OpenCode side by side.
- Free OSS + BYO subscriptions vs Devin's platform fee + per-ACU billing.
- Markdown specs in git on your machine vs hosted workspace.
- Local-first execution; no cloud egress.

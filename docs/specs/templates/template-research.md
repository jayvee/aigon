# Template & Site Research — Executive Portfolio

> Compiled from 6 parallel research agents. All findings verified via web search (Feb 2026).

## Owner's Favourites

- **[Magic Portfolio](https://demo.magic-portfolio.com/)** — **Current top pick.** Clean design system (Once UI), about/CV, projects, blog (MDX), gallery. Next.js. Free (CC BY-NC 4.0; commercial use needs Once UI Pro license).
- **[Hugo Profile](https://hugo-profile.netlify.app/)** — Liked the project showcase space
- **[Adritian](https://adritian-demo.vercel.app/)** — Liked the overall feel and polish (Hugo-only, no Astro port; free, Bootstrap-based, dark/light, blog, multi-language, testimonials, view transitions)
- **[Case](https://case.erland.me)** — Case-study-first structure (Astro, free)

## Framework Recommendation (Updated)

**Owner's current preference: Magic Portfolio (Next.js + Once UI)**

Magic Portfolio is a strong fit because:
- Clean, polished design system via Once UI — closest to Adritian's feel
- About/CV page, project showcase, blog (MDX), gallery — covers the core content sections
- Next.js gives flexibility to add richer interactivity later if needed
- Static export mode (`output: 'export'`) works on Cloudflare Pages
- Active development, good documentation

**Things to note:**
- License is CC BY-NC 4.0 — free for personal/non-commercial use. If the site is used commercially (e.g., consulting lead gen), you'd need the Once UI Pro license
- Next.js is more complex than Astro/Hugo — more moving parts, larger build output
- No testimonials section out of the box (would need to add)
- Cloudflare Pages build config: `npx next build`, output directory `out/`

**Fallback options if Magic Portfolio doesn't work out:**
1. **Astro Case** (free) — case-study-first, simplest stack
2. **Tailwind Spotlight** ($79) — premium polish, also Next.js
3. **Adritian** (Hugo, free) — owner liked the overall feel

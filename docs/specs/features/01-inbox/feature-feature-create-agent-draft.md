# Feature: feature-create-agent-draft

## Summary

Allow aigon feature-create to optionally launch an agent (e.g. --draft cc) to flesh out the spec from a short description. The agent writes user stories, acceptance criteria, and technical approach sections in the bare template. Feature stays in inbox for user review. Closes the gap between 'describe a feature' and 'feature spec ready to prioritise' without requiring the user to be inside an agent first.

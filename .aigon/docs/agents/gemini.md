# Gemini CLI — Retired

The **Gemini CLI** agent (`gg`) was deactivated on 2026-06-18 when Google shut down the standalone Gemini CLI.

**Use Antigravity instead:** see [antigravity.md](./antigravity.md) (`ag`).

Historic `feature-*-gg-*` telemetry and transcripts remain readable via `aigon feature-transcript` and the dashboard analytics views — the registry keeps `gg` for display metadata only.

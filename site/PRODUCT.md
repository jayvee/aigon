## Initial Scope

The first shipped version is a polished landing page with:

1. A fixed minimal nav
2. A dominant name/brand-led hero
3. A concise framing section
4. A clear contact path

## Technical Foundation

- Stack: plain HTML + CSS only
- Build/deps: none (no framework, no npm, no bundler)
- Delivery: static files served directly
- Hosting: Cloudflare Pages connected to GitHub
- Deployment: push to `main` triggers automatic deploy
- Domain: custom domain configured in Cloudflare Pages with HTTPS

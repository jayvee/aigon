---
title: "feature-autopilot (removed)"
description: "Removed — use feature-autonomous-start instead"
---

`feature-autopilot` has been removed. Use [`feature-autonomous-start`](/docs/reference/commands/feature/feature-autonomous-start) instead.

```bash
# Before (no longer works)
aigon feature-autopilot 42 cc gg

# After
aigon feature-autonomous-start 42 cc gg
```

See the [Autonomous Mode guide](/docs/guides/autonomous-mode) for full documentation.

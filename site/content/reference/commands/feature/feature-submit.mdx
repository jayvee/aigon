---
title: "feature-submit (removed)"
description: "Removed — submission is now signaled via agent-status"
---

`feature-submit` has been removed. Implementation submission is now signaled
automatically by the shell trap wrapper around [`feature-do`](/docs/reference/commands/feature/feature-do),
or manually via `aigon agent-status submitted`.

```bash
# Before (no longer works)
aigon feature-submit

# After — automatic on clean exit of feature-do
aigon feature-do 42

# Or signal manually
aigon agent-status submitted
```

See the [Drive Mode guide](/docs/guides/drive-mode) for the full submission
flow.

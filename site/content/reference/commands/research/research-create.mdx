## Synopsis

```bash
aigon research-create <topic-name>
```

## Shortcuts

- `/arc` (slash command) · `aigon arc` (CLI)

## Description

Create research \<name\> - creates topic in inbox

## Usage

```bash
# Slash command (Claude Code / Gemini)
/aigon:research-create <topic-name>

# CLI
aigon research-create <topic-name>
```

## Spec frontmatter

The inbox topic is created from the research template with YAML frontmatter including **`complexity`**, same idea as features. Model/effort defaults when you **research-start** are resolved from that label plus each agent’s ladder and config — not from model names in the spec. See [Agents: Spec complexity and model defaults](/docs/reference/agents#spec-complexity-and-model-defaults).

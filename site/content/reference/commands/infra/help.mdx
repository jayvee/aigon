## Synopsis

```bash
aigon help
```

## Shortcuts

- `/ah` (slash command) · `aigon ah` (CLI)

## Description

`aigon help` prints a grouped summary of all available commands — feature, research, feedback, and infrastructure — directly in the terminal. It is the quickest way to discover command names and shortcuts without leaving your session.

For detailed documentation on a specific command, see its reference page in this section or pass the command name to `aigon help <command>` (if supported by your version).

## Usage

```bash
# Print all commands
aigon help

# From an agent session
/aigon:help
```

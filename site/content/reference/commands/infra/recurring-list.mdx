---
title: recurring-list
description: Show all recurring feature templates with their schedule, last created period, and due/not-due status.
---

List all recurring templates and whether they're due for the current ISO week or quarter.

## Usage

```bash
aigon recurring-list
```

## Example output

```
📅 Recurring features (week 2026-W17 · quarter 2026-Q2)

Slug                           Schedule   Last period  Status
--------------------------------------------------------------
weekly-dep-sweep               weekly     2026-W17     ✅ done
weekly-docs-gap-scan           weekly     2026-W16     🔴 DUE
weekly-agent-matrix-benchmark  weekly     —            🔴 DUE
quarterly-agent-matrix-qualit  quarterly  2026-Q1      ✅ done
```

## Status column

| Status | Meaning |
|--------|---------|
| `✅ done` | An open instance exists this period, or one was already created |
| `🔴 DUE` | No open instance and not yet created this period — will fire on next `recurring-run` |

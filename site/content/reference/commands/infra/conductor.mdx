## Synopsis

```bash
aigon conductor <start|stop|status|add|remove|list> [path]
```

## Description

**Deprecated** — `aigon conductor` is the old name for `aigon dashboard`. All subcommands are aliased:

```bash
aigon conductor start   →  aigon dashboard start
aigon conductor add     →  aigon dashboard add
aigon conductor status  →  aigon dashboard status
```

Use [`aigon dashboard`](/docs/reference/commands/infra/dashboard) instead.

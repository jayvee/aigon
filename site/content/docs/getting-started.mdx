title: Getting Started
description: Install Aigon and run your first spec-driven feature loop.
---

## Installation

```bash
git clone https://github.com/jayvee/aigon.git
cd aigon
npm install
npm link
```

## Initialize a project

```bash
cd /path/to/your/project
aigon init
aigon install-agent cc gg cx cu
```

## Your first feature

In Claude Code:

```bash
/aigon:feature-now dark-mode
```

This creates a feature spec, sets up the workspace, and starts implementation — all in one command.

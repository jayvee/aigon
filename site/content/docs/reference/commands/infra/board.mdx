## Synopsis

```bash
aigon board [--list] [--features] [--research] [--active] [--all] [--inbox] [--backlog] [--done] [--no-actions]
```

## Shortcuts

- `/ab` (slash command) · `aigon ab` (CLI)

## Flags

- `--list`
- `--features`
- `--research`
- `--active`
- `--all`
- `--inbox`
- `--backlog`
- `--done`
- `--no-actions`

## Description

Show Kanban board or list view

## Usage

```bash
# Slash command (Claude Code / Gemini)
/aigon:board [--list]

# CLI
aigon board [--list]
```

# ff-research-start

Run this command followed by the Research ID.

```bash
ff research-start {{args}}
```

This moves the research topic from `02-backlog/` to `03-in-progress/`.

Read the topic file and begin your research. Document findings in the topic file or create additional notes.

When done: Run `ff research-done {{args}}`

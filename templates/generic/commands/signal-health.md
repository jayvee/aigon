# aigon-signal-health

Show per-agent lifecycle signal reliability from `.aigon/telemetry/signal-health/`.

```bash
aigon signal-health {{ARG_SYNTAX}}
```

Useful filters:

```bash
aigon signal-health --agent cx
aigon signal-health --since 2026-04-01
aigon signal-health --entity-type research --json
```

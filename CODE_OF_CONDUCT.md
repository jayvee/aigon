# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

The full text is available at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — whether through issues, pull requests, discussions, or any other channel — you agree to abide by its terms.

## Reporting

Concerns, reports, or questions about conduct can be sent to:

**conduct@senlabs.ai**

All reports will be reviewed and investigated promptly and fairly. The project maintainers are obligated to respect the privacy and security of the reporter of any incident.
